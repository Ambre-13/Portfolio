# E‑Girl Neon Portfolio

Minimal portfolio demo with dark background and neon pink/purple palette.

Ouvrir `index.html` dans un navigateur pour voir le rendu.

Fichiers:
- index.html
- style.css
- scripts.js

Ajustements recommandés: jouer sur les variables CSS (couleurs, intensité du glow), ajouter images et contenus réels pour personnaliser.
